roBrowser

Open Source reproduction of Ragnarök Online for the Web Browser

Author: Vincent Thibault
Site: http://www.robrowser.com/
Source: http://www.github.com/vthibault/roBrowser/

roBrowser is an open source project based on the game Ragnarök Online.
It’s not affiliated in any way with Gravity.

The concept is to reproduce the game using awesome new web technologies (HTML5, Javascript, WebGL) to bring it to Web Browsers.

Since roBrowser runs in a web browser, it also means it’s cross-platform and can run with multiple OSes (Windows, Linux, Macintosh, ...) instead of the original client which was limited to Windows.
It should also run fine on other devices ( televisions, computers, smartphones, tablets) if the device supporst openGL ES 2.0 and is powerful enough to support the game.
